    The Legend of Zelda (NES Game Unity Remake)

  • Released Date: September 2022
  • Software Used: Unity, C#, Visual Studio, GitHub, Jira

    • The Legend of Zelda (NES Game Unity Remake) is a Unity recreation version of the first dungeon: The Eagle from The Legend of Zelda. This was the first major project of EECS 494 and was developed in three weeks with a partner.

    We were also assigned to create our own custom level with a new game mechanic. We have come up with a new item called UltraHand. Highly inspired by UltraHand, a toy that is designed and released by Nintendo during the late 1960s, UltraHand in our game can drag non-static objects like enemies to Link. On the other hand, Link will be pulled towards the object if used on a static object like walls.
